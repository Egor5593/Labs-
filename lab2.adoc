== Цель

Цель отчета – какие типы данных бывают, их преобразования, указатели.

== Основные целочисленные типы данных 

[cols="2,2,6",options="header"]
|===
| Тип | Размер | Описание
| bool | 1 байт | Логическое значение true / false.
| char | 1 байт | Символ ASCII. Используется в строках.
| unsigned char | 1 байт | Беззнаковый символ (альтернатива `std::byte`).
| int | 4 байта | Стандартное целое число.
| unsigned int | 4 байта | Беззнаковое целое.
| float | 4 байта | Число с плавающей точкой.
| double | 8 байт | Число двойной точности.
|===

=== Модификаторы типов

* short int — 2 байта
* unsigned short int — 2 байта
* long int — 8 байт
* unsigned long int — 8 байт
* long double — 8 байт

== Правила определения размеров типов

1 <= sizeof(char) <= sizeof(short) <= sizeof(int) <= sizeof(long)
1 <= sizeof(bool) <= sizeof(long)
sizeof(float) <= sizeof(double) <= sizeof(long double)
sizeof(T) == sizeof(signed T) == sizeof(unsigned T)

== Преобразования типов

=== Неявное преобразование

*Происходит автоматически*:

[source,cpp]
----
float f = 10; // int → float
----

Может произойти потеря данных.

=== Явное преобразование

*Требует явного указания*:

[source,cpp]
----
int x = 10;
float f = static_cast<float>(x);
----

Нужно для предотвращения нежелательной потери данных или неожиданного поведения программы.

== Операторы преобразования типов

[cols="2,6",options="header"]
|===
| Оператор | Назначение
| static_cast | Безопасное преобразование совместимых типов
| reinterpret_cast | Переопределение типа указателя (лучше не использовать)
| dynamic_cast | Преобразование между типами с виртуальными функциями
| const_cast | Изменение константности объекта
|===

== Указатели

Указатели — это переменные, которые хранит адрес другой переменной в памяти. С помощью указателей можно косвенно обращаться к данным и управлять ими.

== Основные операции с указателями

С указателями можно выполнять следующие действия:

=== Присваивание адреса переменной

[source,cpp]
----
int x = 5;
int* p = &x;  // указатель p получает адрес переменной x
----

=== Разыменование указателя 

Позволяет получить значение, на которое указывает указатель.

[source,cpp]
----
int y = *p;   // y получает значение, хранящееся по адресу p (то есть значение x)
----

=== Взятие адреса

Оператор получения адреса переменной.

[source,cpp]
----
int* q = &x;  // q получает адрес переменной x
----

=== Арифметика указателей

Указатели можно увеличивать или уменьшать — они перемещаются по элементам массива.

[source,cpp]
----
int arr[] = {10, 20, 30};
int* ptr = arr;
ptr++;  // теперь указывает на arr[1]
----

=== Сравнение указателей

Можно сравнивать два указателя между собой.

[source,cpp]
----
if (ptr1 == ptr2) {
    // указатели равны
}
----

=== Присваивание нулевого значения 

[source,cpp]
----
int* p = nullptr;  // указатель не указывает ни на один объект
----

== Константные указатели и указатели на константу

Различия между типами указателей, связанными с константностью:

=== Указатель на константу (`const int* p` или `int const* p`)

* Нельзя изменять значение по адресу
* Можно изменять сам указатель

[source,cpp]
----
const int* p = &x;   // или: int const* p = &x;
*p = 10;             // ошибка: нельзя менять значение
p = &y;              // допустимо: можно переназначить указатель
----

=== Константный указатель (`int* const p = &x`)

* Можно изменять значение по адресу
* Нельзя изменить сам указатель (адрес)

[source,cpp]
----
int* const p = &x;
*p = 20;             // допустимо
p = &y;              // ошибка: указатель неизменяемый
----

=== Константный указатель на константу (`const int* const p = &x`)

* Нельзя ни изменять значение, ни переназначать указатель

[source,cpp]
----
const int* const p = &x;
*p = 20;             // ошибка: значение нельзя изменить
p = &y;              // ошибка: адрес менять нельзя
----

== Разыменовывание указателя
Разыменование — это операция получения значения, расположенного по адресу, хранящемуся в указателе.

[source,cpp]
----
int x = 5;
int* p = &x;
int y = *p;  // y = значение x через указатель
----

== Взятие адреса переменной

Оператор & используется для получения адреса переменной.

[source,cpp]
----
int x = 10;
int* p = &x;  // p получает адрес переменной x
----

== Сводная таблица по указателям

[options="header"]
|===
| Операция               | Описание
| *p                   | Разыменование — доступ к значению по адресу
| &x                   | Взятие адреса переменной
| const int* p         | Указатель на константу (значение нельзя изменять)
| int* const p         | Константный указатель (адрес нельзя изменять)
| const int* const p   | Константный указатель на константу
| p++, p--, p + n  | Арифметика указателей (перемещение по памяти)
|===

== Битовые операции

[cols="2,6",options="header"]
|===
| Операция | Выражение
| Установить бит | x |= (1 << n);
| Сбросить бит | x &= ~(1 << n);
| Переключить бит | x ^= (1 << n);
|===

== Унарные операторы

* +x — возвращает само значение без изменений.
* -x — меняет знак на противоположный.

== Почему 17000000 + 1 == 17000000 для float?

Число 17000000 (`1.7e7`) в формате float представляется с точностью около 7-8 значащих цифр.  
При прибавлении 1 изменений в значении не происходит, потому что разница меньше разрешённой точности формата.

== Краткий вывод 

* Размер типов зависит от архитектуры и компилятора.
* Явные преобразования — контроль за безопасностью кода.
* Осторожно с разностью указателей и использованием reinterpret_cast.
* Помнить о точности при работе с числами с плавающей точкой (`float`).
